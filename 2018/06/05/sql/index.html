SQL语句及其种类

  • DDL 数据定义语言
    CREATE 创建数据库和表等对象
    DROP 删除数据库和表等对象
    ALTER 修改数据库和表等对象的结构
  • DML 数据操作语言
    SELECT 查询表中数据
    INSERT 向表中插入数据
    UPDTE 更新表中数据
    DELETE 删除表中数据
  • DCL 数据控制语言
    COMMIT 确认对数据库中的数据进行修改
    ROLLBACK 取消对数据库中的数据进行变更
    GRANT 赋予用户操作权限
    REVOKE 取消用户的操作权限

SQL的基本书写规则

不区分关键字大小写
常数的书写方式是固定的
单词需要用半角空格或者换行来分隔
